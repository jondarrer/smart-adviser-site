# smart-adviser-site

The website for [smartadviser.co.uk](https://www.smartadviser.co.uk).

## Development

### Install

```bash
npm i
```

### Run locally

```bash
npm start
```

You can also try out how it will work when deployed to Firebase with:

```bash
npm run start:fb
```

Which will run the following commands:

```bash
npm run build
firebase serve --only hosting
```

### Test

```bash
npm test
```

### Build

```bash
npm run build
```

### Deploy

Deploy with:

```bash
git push
```
